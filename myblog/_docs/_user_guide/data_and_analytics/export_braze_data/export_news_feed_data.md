# News Feed Data

When analyzing the performance of a specific News Feed card, you can export the data found in the Performance Breakdown graphs:

- Impressions by Date
    - Clickthrough Rate
    - Impressions
    - Unique Impressions
- Clicks by Date
    - Clickthrough Rate
    - Clicks
    - Unique Clicks
